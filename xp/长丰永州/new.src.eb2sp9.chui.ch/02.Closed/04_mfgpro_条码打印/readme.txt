2.条码标签存放位置:
要先在通用代码维护好,
例如:
	mgcodemt.p 2+                  36.2.13 通用代码维护                   10/06/07
	+------------------------------------------------------------------------------+
	|                字段名称: BARCODE                                             |
	|                      值: LabelsPath                                          |
	|                    说明: /app/bc/labels                                      |
	+------------------------------------------------------------------------------+

然后把标签文件复制到该目录下.

3.原材料条码打印: xxbcrp002.p
	(按采购收货记录)
	格式:件号+$+收货日期+供应商代码+2位流水号
	文件名:xsposn10
	流水号:当天此part所有收货记录的排序号

4.半成品条码打印:xxbcrp001.p
	(按输入的计划员,查找R状态工单)
	格式:件号+$+下达日期+2位流水号
	文件名:xswosn10
	流水号:当天此part所有R状态wo的排序号
